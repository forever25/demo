### js引入方式

1. 写在html 中 

   ```html
   <script>
   	var a = 12;
       // 这样写不利于维护，可扩展性差
   </script>
   ```

   

2. 外部引入

   ```html
   <script src="url" type="text/javascript">
        // type 属性标识脚本语言类型 mime 类型
       // type h5 开始省略type属性
   </script>
   ```
